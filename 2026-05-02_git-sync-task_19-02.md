# 2026-05-02 Git 多仓库同步任务

## 目标
创建自动化脚本，一键推送三个仓库：
- QClaw 工作区 → GitHub
- D:\busd → Gitee + GitHub
- D:\usdt → Gitee

## 最终方案
使用 Node.js + spawn（非 simple-git），解决 PowerShell 参数拆分问题

## 关键修复
1. **代理逻辑修正**：Gitee 直连（国内），GitHub 走代理 127.0.0.1:7890（国外）
2. **spawn 替代 execSync**：解决 commit message 含空格被拆分的问题
3. **时间格式**：用 `YYYY-M-D H-mm-ss` 替代 `toLocaleString()`，避免 `/` 被 PowerShell 当路径分隔符

## 脚本位置
`C:\Users\Administrator\.qclaw\workspace\scripts\git-sync-all.js`

## 使用方法
```bash
node scripts/git-sync-all.js          # 正式推送
node scripts/git-sync-all.js --dry-run  # 预览模式
```

## 结果
三个仓库全部推送成功 ✅
